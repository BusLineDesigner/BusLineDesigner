    关于 Bus Line Designer

    本网站（Bus Line Designer，简称 BLD）是一个免费的在线工具，旨在帮助用户便捷地创建、展示和处理公共交通线路信息。

    关于地图服务

    BLD 本身不提供地图服务。目前，本站的地图相关功能（如地图展示、路径计算、站点搜索等）完全依赖第三方平台高德地图的 API 接口实现。

    因此，您需要自行获取高德地图的 API Key 并配置后方可使用地图功能。如果您不进行配置，所有依赖于地图的功能将无法使用，例如，地图面板将显示为空白。

    使用高德地图 API 的步骤与法律责任

    1. 注册与遵守协议

    如果您决定使用地图功能，请首先：
    • 在高德地图开放平台注册开发者账号。
    • 务必仔细阅读并严格遵守 《高德地图开放平台服务协议》 及其相关规范。

    特别风险提示：高德地图对“个人”与“企业”认证开发者有不同的使用权限限制。您必须根据自身属性（商用与否）选择并遵守对应的服务条款。任何因您违反高德地图协议而导致的法律责任（如索赔、封禁等），均须由您自行承担。BLD（本站）作为技术接口提供方，无法也无需对此负责。

    2. 获取并配置 Key

    在确保理解并同意上述协议后，请按以下步骤操作：

    1. 登录高德开放平台，进入「应用管理」→「我的应用」，点击「创建新应用」。
    2. 创建后，点击该应用右侧的「添加」，为应用添加Key，「服务平台」选择“Web端（JSAPI）”。
    3. 添加成功后，您将获得一个「Key」和一个「安全密钥」（均为32位字符串）。
    4. 打开 BLD 的「设置」→「通用设置」面板，找到「API Key + 安全密钥」项。
    5. 将高德提供的 Key 和安全密钥依次填入输入框，中间用一个空格隔开。
    6. 保存设置并刷新页面，地图功能即可生效。
    3. 最终确认

    当您在 BLD 中配置您的高德 API Key，即代表：
    1. 您已阅读并同意高德地图的相关协议。
    2. 您确认您的使用行为是合规的。
    3. 您理解并同意，BLD 仅提供了一个技术界面，您通过本站对高德 API 的任何调用行为，均视为您本人的行为。

    关于未来
